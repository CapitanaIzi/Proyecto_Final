# Backend – API con Express, Firestore y JWT

Este proyecto implementa una API REST con:
* Arquitectura en capas (routes, controllers, services, models)
* Conexión a Firebase Firestore para persistencia de datos
* Autenticación y autorización mediante JWT
* Protección de rutas
* Manejo de errores

## Tecnologías utilizadas
* Node.js
* Express
* Firebase Firestore
* JSON Web Tokens (JWT)
* dotenv
* Thunder Client (para pruebas)
